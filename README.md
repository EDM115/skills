# `EDM115/skills`
Skills generated for my own purpose, created because nobody made them before

## Usage
### Select skills
```zsh
pnpm dlx skills add EDM115/skills
```

### Install a specific skill
```zsh
pnpm dlx skills add EDM115/skills --skill pyrofork-best-practices
```
